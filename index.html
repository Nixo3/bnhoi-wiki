Balanced Never Heard Of It

A total conversion mod inspired by Better Than Wolves and MITE "You don't walk to a tree and carve it into a toolbench."

What is BNHOI?

Balanced Never Heard Of It (BNHOI) is a total conversion mod for Minecraft 1.21 that significantly reworks game progression, survival mechanics, and overall difficulty.

The goal of the mod is to make survival feel earned. Primitive tools, delayed automation, and meaningful player decisions form the core pillars of its design.

Main Changes

  • Punishment For low health and hunger.
  • Player metabolism changed to make hunger drain way faster
  • Impossible to harvest some blocks with wrong tools
  • Added new items that are crucial in progression
  • Added new mobs
  • Added new blocks
  • Added new animal behavior
  • Added changes to hostile mobs
  • Replaced various recipes
  • Reworked some vanilla textures

    • Installation

    In order to play the mod, you have to download the mod itself and some dependencies such as:

  • GeckoLib for 1.21
  • Fabric Api for 1.21

    • Balanced never heard of it is found on On Modrinth as well as the dependencies mentioned earlier.

    Note: GeckoLib is only needed for BNHOI 1.3.0 and and later, at this time 1.3.1 may already be out.
